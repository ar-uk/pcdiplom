## MATCHING ALGORITHM WEIGHTS

┌─────────────────────────────────────────────────────────────────────────────┐
│ Matching Strategy                          Weight    Example Score          │
├─────────────────────────────────────────────────────────────────────────────┤
│ Exact string match                          1.0      "RTX 4090" = "RTX 4090"│
│ Substring match                             0.95     Name contained         │
│ Manufacturer brand detection                0.2      Intel, AMD, NVIDIA    │
│ Token overlap (Jaccard similarity)          0.5-0.6  Common words/numbers   │
│ Levenshtein edit distance                   0.15-0.2 Typo tolerance        │
│ Specification matching (cores, mem, etc)    0.15     Match 24GB GDDR6X     │
├─────────────────────────────────────────────────────────────────────────────┤
│ Combined weighted score: 0.0 (no match) → 1.0 (perfect)                    │
└─────────────────────────────────────────────────────────────────────────────┘


## CONFIDENCE SCORE INTERPRETATION

┌──────────────────────────────────────────────────────────────────────────────┐
│ Confidence   │ Action             │ Example                                   │
├──────────────┼────────────────────┼───────────────────────────────────────────┤
│ 0.95+        │ ✅ AUTO-ACCEPT    │ "i9-13900KS" → "Intel Core i9-13900KS"   │
│              │ (use directly)     │ (exact brand+model match)                 │
├──────────────┼────────────────────┼───────────────────────────────────────────┤
│ 0.75-0.95    │ ⚠️  REVIEW        │ "Core i7" → "Intel Core i7-13700K"        │
│              │ (verify specs)     │ (brand+model match, minor variants)      │
├──────────────┼────────────────────┼───────────────────────────────────────────┤
│ 0.50-0.75    │ 🔍 MANUAL CHECK   │ "7950X" → "AMD Ryzen 9 7950X"             │
│              │ (needs review)     │ (model only, no brand)                    │
├──────────────┼────────────────────┼───────────────────────────────────────────┤
